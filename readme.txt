# FX-IB-Marketing
 サイトデータについて

このフォルダは FX-IB-Marketing
サイトのサーバー移転に必要な公開データ一式です。
本書では使用しているテンプレート、画像、スクリプトの権利情報をまとめています。

----------------------------------------------------
■ フォルダ構成
/public_html
  /css/       ← スタイルシート
  /images/    ← 画像データ（iStock 無料素材・オリジナル画像 等）
  /js/        ← メインスクリプト・プラグイン
/readme.txt   ← 本説明書
/SECURITY.md  ← セキュリティポリシー

----------------------------------------------------
■ テンプレートについて
- 使用テンプレート: Styleshout 提供「Monica」
- ライセンス: 無料で個人・商用利用可能
- 条件: フッターなどに Styleshout へのクレジットリンクを設置する必要あり
  （例：「Design by Styleshout」）
- クレジット削除を希望する場合は、Styleshout 公式にて 1 ドメインにつき 10 USD で除去可
  （支払方法: PayPal / 公式サイト経由）

----------------------------------------------------
■ 画像について
- 一部画像は iStock の公式無料素材を利用
- 商用利用可能なロイヤリティフリー素材で、クレジット表記不要
- ただし「Editorial Use Only」素材や商品化・大量印刷には利用不可
- 長期利用や高解像度が必要な場合は、有料版の購入を推奨

----------------------------------------------------
■ スクリプトについて
- /js/ 内の一部コードは mamewaza 氏が公開しているオープンソーススクリプトを利用
- 著作権は mamewaza 氏に帰属
- ライセンス範囲内で商用利用・改変可能
- 追加費用や著作表示の義務はありません
- 再配布・販売は禁止

----------------------------------------------------
■ セキュリティについて
- /SECURITY.md にサポート対象バージョンと脆弱性報告方法を記載
- サーバー移転後は PHP やライブラリ等のバージョンが現行サポート対象かご確認ください

----------------------------------------------------
■ ご不明な点がある場合
- サイトの運用や更新に関してご質問があれば、制作者までお問い合わせください
